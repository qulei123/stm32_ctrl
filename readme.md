

## 硬件环境：rk3399、stm32f103
## 开发工具：gcc、keil
## 项目简介：研制了一台桌面机器人，使用rk3399 + stm32的结构，采用uart进行通信。rk3399搭载安卓系统使用APP进行语音控制和内容显示, stm32负责一些数据采集和动作控制等。

## stm32软件功能
1. 完成stm32与rk3399通信协议的制定和编码；
2. 完成stm32 uart舵机通信控制、ad电压采集、onewire温度采集、
   io输入输出控制；
3. 完成stm32事件处理模块、定时器模式、io输入处理模块，
   方便程序模块化设计和后期维护；
4. 完成了一个简单的shell功能及log打印功能，便于自测。
